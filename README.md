# Basil-s-Flight
Unity game about flying planes and having very good moments

Player Controls:
- WASD - Move around
- Mouse - Look around
- Space - Jump
- Left-Shift - Sprint
- F - Interact

Aircraft Controls:
- WS - Change Pitch
- AD - Change Roll
- QE - Change Yaw
- F - Exit Aircraft

Plot:\
*Yay! Playtime! Flying o'er New York and Halifax! It will be fun! - 2nd Lt. Basil Avery, 2124 A.D.*
