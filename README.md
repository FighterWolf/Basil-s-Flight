# Basil-s-Flight
Unity game about flying planes and having very good moments

Player's Ground Controls:
- WASD - Move around
- Mouse - Look around
- Space - Jump
- Left-Shift - Sprint
- F - Interact/Enter Vehicle

Aircraft Controls:
- WS - Change Pitch
- AD - Change Roll
- QE - Change Yaw
- F - Exit Aircraft

Plot:\
*Yay! Playtime! Flying o'er New York to Halifax! It will be fun! - 2nd Lt. Basil Avery, 2124 A.D.*
